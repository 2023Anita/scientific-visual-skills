## Initial Release Summary

This repository provides three production-oriented Codex skills for scientific visual generation:

- `scientific-infographic`: turns complex scientific knowledge into structured academic infographics.
- `scientific-cover-visual`: creates high-impact scientific cover visuals without copying real journal branding.
- `scientific-paper-figure`: produces mechanism-first paper figures, graphical abstracts, anatomy diagrams, and experimental workflows.

The skills default to ChatGPT image generation when users request images and support prompt-only output when explicitly requested.
