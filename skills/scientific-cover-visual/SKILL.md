# Scientific Cover Visual

用于生成科研杂志封面图、科研主视觉、期刊封面风格图、PPT 封面、公众号封面、科学概念海报。

核心原则：**先提炼科学概念，再做强主视觉**。封面图的目标是冲击力、记忆点和科研气质，不是解释所有机制。

## 默认行为

- 如果用户要“生成图”“做封面”“出一张封面”“画主视觉”，直接调用可用的 ChatGPT 生图工具生成最终图像。
- 只有用户明确说“只要提示词 / prompt / 不要生成图”时，才只输出提示词。
- 如果生图工具不可用，输出完整可复制提示词，并说明当前无法直接渲染。
- 可以有虚构刊名，但禁止真实期刊 logo、真实刊名字样、官方版式和机构品牌。

## 输入槽位

尽量从用户请求中提取；缺失时合理推断。

- 封面主题 / 主标题
- 副标题，可空
- 研究领域：生物医学、材料、纳米、化学、能源、环境、神经科学、免疫、肿瘤、外科、芯片、器件等
- 研究对象：细胞、组织、器官、分子、材料、晶格、纳米颗粒、芯片、器件、病原体、生态系统等
- 核心机制 / 视觉概念：一句话
- 关键视觉元素：3-6 个
- 封面类型：虚构期刊封面、无刊名纯主视觉、科研海报封面、PPT 封面、公众号封面
- 语言：中文、英文、中英混排、不要文字
- 画幅比例：1:1、4:5、3:4、16:9、5:2、3:2
- 文字需求：极少文字、只要标题、标题+副标题、不要文字
- 配色方案
- 版式偏好
- 禁用元素

## 概念提炼

如果主题较长，先压缩为一个短核心词或短语作为主视觉标题。完整主题可作为副标题或小字信息。不要把长标题整句粗暴放大。

封面不是机制图，但必须能暗示研究对象、科学机制或学术主题。

## 封面版式策略

根据主题自动选择，不要固定为“顶部大刊名 + 中间物体 + 底部流程”。

- 单一强主体型：疫苗、材料、器官、器件、芯片、纳米结构。
- 微观宇宙型：细胞、分子、晶格、病毒、材料微结构形成宏大空间感。
- 概念隐喻型：桥、光束、屏障、护盾、网络、裂缝、通道等。
- 结构剖面型：透明剖面、层级切面、组织截面、材料截面。
- 动态作用型：粒子迁移、信号传导、免疫攻击、能量转换、材料响应。
- 极简高级型：大留白、单主体、少量高亮色。
- 高冲击封面型：强光影、高对比、深色背景、鲜明强调色。

## 配色系统

- Nature 系克制高级：冷灰、雾蓝、象牙白、深墨蓝；少量珊瑚红、金黄、深蓝。
- Science 系理性高级：青灰、蓝灰、深蓝；少量砖红、芥黄、深青。
- Cell 系生命科学高对比：深蓝紫、靛蓝、黑蓝、紫红、青蓝；少量荧光橙、亮青。
- Advanced Science / Advanced Materials 感：深海蓝、电青、品红、紫、金属黑；局部高亮荧光色。
- 黑白灰 + 单一强调色：红、蓝、绿或金小面积点缀。
- 浅色高级医学封面：象牙白、浅肉粉、淡蓝灰、浅紫、雾粉。

## 文字系统

- 文字少而精，是视觉系统的一部分。
- 可使用虚构刊名，如 `SCIENCE VISUAL REVIEW`、`ADVANCED LIFE SYSTEMS`、`MOLECULAR FRONTIERS`、`BIOMEDICAL HORIZONS`。
- 刊名不是必须出现，不要每次固定放顶部。
- 可采用左上角小刊名、侧边刊名、底部信息栏、无刊名纯主视觉。

## 生成提示词结构

调用生图工具时，把提示词组织为：

1. 角色：顶级科研封面视觉总监、学术期刊封面设计师、科学概念艺术导演。
2. 任务：围绕主题生成“顶级科研期刊封面气质 + 原创主视觉概念 + 强视觉冲击”的封面图。
3. 输入槽位：主题、领域、对象、核心机制、关键元素、封面类型、语言、比例、文字需求、禁用元素。
4. 主视觉策略：选择一种非模板构图。
5. 视觉要求：主视觉强、概念清楚、配色有层次、文字克制。
6. 禁止项：真实期刊 logo、真实刊名、官方版式、普通蓝紫科技海报、电商广告感、医美广告感、PPT 模板感。
7. 最终标准：一眼有冲击力，主视觉清楚原创，科学概念可被感知。

## 质量门槛

生成前自检：

- 是否有一个清楚、震撼、可记忆的核心视觉？
- 是否避免复制任何真实期刊或固定封面结构？
- 科学对象和主题是否能被感知？
- 文字是否足够克制？
- 是否比普通科研海报更像真正封面作品？

如果不满足，先修正提示词再调用生图工具。
