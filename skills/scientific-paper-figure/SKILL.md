# Scientific Paper Figure

用于生成论文插图、Graphical Abstract、作用机制图、实验流程图、解剖结构图、手术步骤图、病理机制图、工作原理图、技术路线图。

核心原则：**科学结构和因果路径优先**。论文插图的目标是准确表达机制、结构和过程，不是单纯做漂亮科技图。

## 默认行为

- 如果用户要“生成图”“做论文插图”“画机制图”“做 graphical abstract”，直接调用可用的 ChatGPT 生图工具生成最终图像。
- 只有用户明确说“只要提示词 / prompt / 不要生成图”时，才只输出提示词。
- 如果生图工具不可用，输出完整可复制提示词，并说明当前无法直接渲染。
- 不使用真实期刊 logo、真实品牌、真实机构标识。

## 输入槽位

尽量从用户请求中提取；缺失时合理推断。涉及医学、解剖、手术、病理机制时，如果关键信息不足且容易造成明显错误，先提出 1-2 个必要问题。

- 图像类型：Graphical Abstract、作用机制图、工作原理图、论文主图、实验流程图、技术路线图、解剖结构图、手术步骤图、病理机制图
- 研究主题
- 研究领域：生物医学、材料、纳米、化学、能源、神经科学、免疫、肿瘤、外科、解剖等
- 研究对象：细胞、组织、器官、解剖结构、分子、材料、器件、芯片、病原体等
- 核心机制 / 核心内容：一句话
- 输入端，可空
- 过程机制，可空
- 输出结果，可空
- 关键视觉元素：3-10 个
- 结构形式：自动判断、左中右流程、上中下流程、A→B→C→D、主体+局部放大窗、微观-中观-宏观、解剖剖面、多面板图组
- 语言：中文、英文、中英混排、不要文字
- 画幅比例：1:1、4:5、3:4、16:9、5:2、3:2
- 文字需求：少量标签、完整标注、不要文字
- 配色方案
- 禁用元素

## 科学叙事结构

根据主题自动选择：

- 输入 → 过程 → 输出：Graphical Abstract、工作原理图。
- A → B → C → D：作用机制图、实验流程图、手术步骤图。
- 主体 + 局部放大窗：论文主图、结构解释图、细胞机制图、解剖细节图。
- 微观 → 中观 → 宏观：材料、生物医学、组织工程、纳米医学。
- 左侧问题 / 中间方法 / 右侧结果：论文图形摘要、技术路线图。
- 解剖剖面 / 结构分层：器官结构图、病理机制图、手术说明图。

画面必须有明确方向。箭头、流线、编号、标签必须服务逻辑，不可乱用。

## 论文插图要求

- 图形清晰：关键结构可辨认，不要抽象到看不懂。
- 标签准确：短、清楚、功能化，避免大段文字。
- 机制明确：递送、结合、催化、释放、激活、抑制、修复、检测、传输、切除、缝合、吻合、植入等过程要用路径表现。
- 局部放大：微观机制、解剖细节或关键操作点应使用局部放大窗。
- 科学可信：不要为了好看牺牲基本科学逻辑；解剖结构、器官位置、组织层级、手术步骤和机制路径要尽量合理。

## 配色系统

- Nature 系克制机制：冷灰、雾蓝、象牙白；淡粉、浅绿、淡紫灰；少量珊瑚红、深蓝、金黄。
- Science 系理性机制：中性蓝、青灰、蓝灰；浅紫、浅青、豆沙粉；少量砖红、芥黄、深青。
- Cell 系生命科学高对比：深蓝紫、靛蓝、紫红、青蓝、粉橘、冷白；少量荧光强调色。
- 医学图谱浅色：浅肉粉、器官红、象牙白、浅蓝、浅灰、浅紫。
- 外科解剖：肉粉、组织红、脂肪黄、象牙白；深红标注血管，深蓝标注神经，橙色标注手术步骤。

## 文字与标签规则

- Graphical Abstract 可使用短标题和阶段标签。
- 机制图必须有关键步骤标签。
- 解剖图必须保证结构标签清楚但不拥挤。
- 手术步骤图可使用编号、箭头和短句说明。
- 不写大段解释性文字。
- 如果用户选择“不要文字”，尽量通过视觉结构表达机制。

## 生成提示词结构

调用生图工具时，把提示词组织为：

1. 角色：顶级论文插图设计师、Graphical Abstract 设计师、科研机制图专家、解剖图谱设计师。
2. 任务：围绕主题生成“论文级科学准确性 + 机制可视化 + 清晰因果路径”的科研论文插图。
3. 输入槽位：图像类型、主题、领域、对象、核心机制、输入端、过程、输出、关键元素、语言、比例、文字需求、禁用元素。
4. 结构逻辑：选择科学叙事结构。
5. 视觉要求：机制路径清楚、结构可辨、局部放大、标签准确、箭头有方向。
6. 禁止项：漂亮但无机制、箭头乱飞、结构不清、过度发光、游戏 CG 感、商业广告感、标签不可读、科学结构明显错误。
7. 最终标准：像论文高质量主图、机制图、解剖图或图形摘要。

## 质量门槛

生成前自检：

- 研究对象或解剖结构是什么？
- 输入端或起始条件是什么？
- 中间发生了什么机制、过程、变化或操作步骤？
- 最终输出什么结果？
- 关键机制发生在哪个位置、结构、细胞、材料或系统中？
- 箭头和标签是否服务逻辑？
- 是否避免为了好看牺牲科学可信度？

如果不满足，先修正提示词再调用生图工具。
