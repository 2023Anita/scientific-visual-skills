# Image Asset Policy

This repository is intended to be publishable as an open-source GitHub project. Images included in the repository should therefore be safe to redistribute.

## Allowed Assets

Recommended asset types:

- Original images generated specifically for this project.
- Diagrams or SVGs created by the repository owner.
- Public-domain images.
- Images with explicit open licenses compatible with the repository license.
- User-owned OpenSea or NFT artwork where the owner has redistribution rights.

## OpenSea / NFT Images

OpenSea is a marketplace. An image appearing on OpenSea does not automatically grant permission to redistribute it in an open-source repository.

Before adding an OpenSea image to this repository, verify:

- You own the asset or have permission to redistribute it.
- The license allows inclusion in a public GitHub repository.
- Attribution requirements are documented.
- The file does not include unauthorized trademarks, portraits, or protected branding.

Place approved OpenSea images under:

```text
assets/opensea/
```

For each image, add a short note with:

- Source URL.
- Asset title.
- Rights holder.
- License or permission basis.
- Date checked.

## Current Repository State
