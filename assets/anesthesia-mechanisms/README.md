# Anesthesia Mechanisms Image Set

This directory contains an original image set generated for this repository with the three scientific visual skills:

- `scientific-cover-visual`
- `scientific-infographic`
- `scientific-paper-figure`

Topic: mechanisms of anesthesia drugs, organized around three clinical goals:

- Hypnosis / unconsciousness
- Analgesia
- Muscle relaxation

The images are demonstration assets for skill documentation and prompt design. They are not clinical guidance, drug dosing guidance, or a substitute for professional medical training.
